## User Journey

**End Users (most people):**
1. Read README.md
2. Follow DEPLOYMENT.md for complete setup
3. Reference CONFIGURATION.md and TROUBLESHOOTING.md as needed

**Advanced Users (VPS deployment):**
1. Read README.md
2. Follow DEPLOYMENT.md for basics
3. Reference deployment/ADVANCED.md for VPS-specific details
4. Use deployment/*.sh scripts for automation
